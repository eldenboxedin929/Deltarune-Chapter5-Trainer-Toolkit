## 📌 Introduction

The Deltarune Chapter 5 Trainer Toolkit provides players options to modify game variables during single-player sessions. This tool functions externally. It reads and writes memory addresses to allow for stat adjustments, item management, and speed control. You do not need programming knowledge to use this trainer.

## 🛠️ Using Features

The trainer interface displays several tabs for different game aspects. Each feature requires the game to be running.

### Health and Stats
Select the "Player" tab to view your current status. You can toggle "Infinite HP" to prevent health depletion during combat. You may also enter custom values for attack, defense, and gold stats. Click the "Apply" button after entering your desired numbers.

### Item Management
Open the "Inventory" tab to see your current items. This section lists items currently in your save file. Select an item from the drop-down menu and choose "Add to Inventory" to place it into your bag. You can set the quantity for each item individually.

### Movement and Speed
The "World" tab contains controls for game pace. Use the slider to adjust movement speed. Higher values allow you to move across the map faster. Use the "Speed Multiplier" to change the pace of character dialogue and animations.

## 💡 Troubleshooting Common Issues

If the tool does not work, check these common fixes:

*   Antivirus software: Some security programs flag external trainers as suspicious. If the tool refuses to start, add the folder to your antivirus whitelist.
*   Game Version: Ensure your trainer version matches your game version. Updates to the game can break compatibility with tools.
*   Administrator Access: If the trainer cannot read stats, the game likely blocks the trainer's access. Always run the software as an administrator.
*   Pathing: Ensure the trainer file is not inside the game installation folder. Keep it in a separate folder in your Documents or on your Desktop.

## 🛡️ Safety and Conduct

This tool operates as an external process. It does not replace any game files or damage your save data. However, modification of game states may cause unexpected glitches. We recommend creating a backup of your save file before you apply changes. Locate your save data in the AppData folder for the game and copy those files to a secure location. 

This toolkit is intended for single-player use only. Use this tool responsibly to enhance your personal exploration experience. Do not attempt to use this tool in any online multiplayer environments if you connect to external servers, as game developers often prohibit these tools in those settings.
